My name is Kurt Dusek, I built this app in response to the code challenge.

From an application architecture standpoint, my overall approach was to build a prototype for a DataProvider class which could be inherited from to easily create a standardized data source from multiple venders. A given exchange can be added to the app by implimenting this data provider class to wrap it's various API calls and converting its order book format to the Order class type.

From there, the DataStore class utilizes dependency injection to consume each instance of a DataProvider and provides business logic to structure individual orders as a complete market for a given symbol. Additionally, the DataStore handles refreshing each DataProvider, searching and identifying overlaping orders, and passing structured market data into React.

In addition to the main requirement I completed all of the extra point items, and have included unit tests with the source code.
